# eth-amsterdam
A repo for EthAmsterdam hackathon - gradual dutch auction in Cairo

## Local Development

### Prerequisites

1. Install Python
1. Install [Poetry](https://python-poetry.org/)
1. Clone this repo

```shell
# Activate a Poetry virtual environment
poetry shell

# Install dependencies
poetry install

# Compile all contracts
nile compile
```
